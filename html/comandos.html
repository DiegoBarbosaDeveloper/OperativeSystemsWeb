Referencia de Comandos

dir

Descripción: Lista archivos y carpetas en el directorio actual

Sintaxis: dir [ruta] [/opciones]

dir /s
dir C:\Users /a
Ejemplo

cd

Descripción: Cambia el directorio actual

Sintaxis: cd [ruta]

cd C:\Users\Documents
cd ..
Ejemplo

mkdir

Descripción: Crea un nuevo directorio

Sintaxis: mkdir [nombre]

mkdir Proyectos
Ejemplo

del

Descripción: Elimina archivos

Sintaxis: del [archivo]

del archivo.txt
Ejemplo

copy

Descripción: Copia archivos

Sintaxis: copy origen destino

copy datos.txt D:\Backup
Ejemplo

move

Descripción: Mueve archivos

Sintaxis: move origen destino

move datos.txt D:\Backup
Ejemplo

ipconfig

Descripción: Muestra configuración de red

Sintaxis: ipconfig [/opciones]

ipconfig /all
Ejemplo

tasklist

Descripción: Lista procesos en ejecución

Sintaxis: tasklist

tasklist
Ejemplo

Get-Help

Descripción: Muestra ayuda sobre comandos y su uso.

Sintaxis: Get-ChildItem [ruta] [-opciones]

Get-ChildItem -Recurse
Get-ChildItem C:\Users -Force
Ejemplo
